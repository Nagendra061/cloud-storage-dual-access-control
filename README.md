# cloud-storage-dual-access-control
A Java Spring Boot application implementing dual-layer access control (RBAC + ABAC) with AES-256 encryption for secure cloud-based file storage and sharing.
Dual Access Control System for Cloud-Based Data Storage

A comprehensive security-focused file storage system that combines Role-Based Access Control (RBAC) 
and Attribute-Based Access Control (ABAC) to provide enhanced protection for cloud-stored files. 
Features include AES-256 encryption, JWT authentication, comprehensive audit logging, and RESTful 
API design. Built with Spring Boot 3.2, Spring Security, and Java 17.
